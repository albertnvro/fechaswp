== Descripción ==
Este plugin permite a los administradores de WordPress asignar fechas aleatorias a las entradas en lotes, con un rango de fechas seleccionable por el usuario. Utiliza un datepicker para facilitar la selección de fechas. Es una herramienta útil para sitios que necesitan actualizar las fechas de publicación de sus entradas de forma masiva y controlada.

== Instalación ==
1. Sube los archivos del plugin al directorio `/wp-content/plugins/cambiar-fechas-entradas-optimizado`.
2. Activa el plugin a través de la pantalla 'Plugins' en WordPress.
3. Ve a la página de configuración del plugin en el menú de administración para utilizarlo.

== Uso ==
1. Ve a la página del plugin en el área de administración de WordPress.
2. Selecciona la fecha mínima y máxima utilizando el datepicker.
3. Haz clic en 'Empezar' para comenzar el proceso de cambio de fechas.
